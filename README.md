# StockWatchOSS
An open-source version of the StockWatch discord bot

Required:
- Node/Docker
- MongoDb instance
- TwelveData Api Key
- MarketAux Api Key

Missing/Limited Features:
- The amount of requests for stock data and news is limited
- New Features will take the longest to reach the OSS version
